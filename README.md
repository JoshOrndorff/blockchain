### Initial Setup

```sh
# Install Rustup
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
./init.sh
```

### Building

```sh
git clone git@github.com:Trinkler/katal-chain.git
cd katal-chain/
./build.sh
# Alternatively run: cargo build --release
cargo build
```

### Run

```sh
./target/debug/katal
```
